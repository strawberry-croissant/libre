**Your hardware. Unrestricted.**

Libre is a lean, privacy-focused operating system environment designed for absolute control. It provides a portable, unrestricted system that runs directly from a USB drive, ensuring zero tracking and maximum performance.

---

## Key Features

- **Zero Restrictions**: Full administrative rights from the start. No software gates or corporate locks.
- **Privacy-First**: Minimized data sharing and local-only configurations.
- **Lightweight**: Optimized system footprint for fast performance and low resource usage.
- **Fast Boot**: Custom configurations designed for rapid boot times from BIOS to Desktop.
- **Run Anything**: Install and run software without administrator blocks or forced updates.

## Supported Platforms

We provide tailored builds for a variety of ecosystems:

- **Windows 11**: The full Libre experience with integrated drivers.
- **Windows 10**: Maximum stability and compatibility for older hardware.
- **Ubuntu**: GNOME-based, beginner-friendly Linux build.
- **Linux Mint**: Lightweight and developer-centric.

## How it Works

1. **Choose**: Select a build tailored to your hardware needs.
2. **Flash**: Use Rufus to create a bootable USB drive.
3. **Boot**: Plug in, boot up, and take control.
