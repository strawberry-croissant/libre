Is Libre free to use?

Yes, Libre is completely free and open-source.

Can I install any software on Libre?

Yes, Libre allows you to install and run any compatible software without restrictions.

Is my data safe while using Libre?

Absolutely. Libre prevents all software-based tracking and monitoring.

How do I update Libre?

The version you downloaded cannot be updated without deleting data. We only recommend updating versions every major release.

When is a major release?

A major release happens every time the first number in the version number is incremented. For example, from 1.0.0 to 2.0.0.

Does Libre support gaming?

Yes, Libre is optimized for performance and can run most games.

What are the system requirements?

Libre is lightweight. You need a 64-bit processor, 2GB of RAM, and at least 20GB of storage space.

What version of the OSes am i using? Is it the latest versions?

We usually update the OSes to their latest release. If we have not updated it, it is probably because we forgot about it. Not to worry, we regularily check for updates and update them as soon as possible.

Why do i have to click so many download links when downloading?

Because github doesn't allow us to upload a zip file for everything, so we just link you to the drivers which you have to click individually.

What are the driver types?

There are two types of drivers: generic and optimized. Generic drivers are for unknown hardware, while optimized drivers are for specific hardware.

Why are there no optimised drivers for Linux distributions?

Linux distributions already come with great drivers, so we don't need to provide optimised drivers.
