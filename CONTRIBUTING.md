# Contributing to Libre

Thank you for your interest in contributing to Libre! We welcome contributions that help make this project better for everyone.

## How to Contribute

### Reporting Bugs

- Use the GitHub Issues tracker to report bugs.
- Provide a clear description of the issue and steps to reproduce it.

### Feature Requests

- We're always open to suggestions! Please open an issue to discuss new features before starting development.

### Pull Requests

1. Fork the repository.
2. Create a new branch for your feature or bug fix.
3. Keep your changes focusedâ€”if you have multiple unrelated changes, use multiple pull requests.
4. Ensure your code follows the existing style and is well-documented.
5. Submit your pull request with a clear description of the changes.

## Development Setup

The project is a static site with some JavaScript. To run it locally:

1. Clone the repository.
2. Open `index.html` in your browser.

## Code of Conduct

Please be respectful and professional in all interactions within this project.
